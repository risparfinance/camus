Camus
=====

Camus is a raw SQL library enabling an ease integration with the
new `Aurora Serverless Data API <https://aws.amazon.com/blogs/aws/new-data-api-for-amazon-aurora-serverless/>`_.
It is a fork of the powerful `Records <https://github.com/kennethreitz/records>`_ library.
